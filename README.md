# Hair Design Review System

美容師向けのヘアデザインレビューシステムです。顧客からのフィードバックを収集・分析し、美容師とオーナーが活用できるダッシュボードを提供します。

## 特徴

- 📊 **スタイリストダッシュボード**: 自分の作品への評価やレビューを確認
- 👥 **オーナーダッシュボード**: スタッフ全体のパフォーマンスを分析・比較
- 🔐 **プライバシー保護**: 個人情報はコードに含まれず、管理画面からアップロード
- 💾 **データ管理**: CSV/画像をアップロードしてlocalStorageに保存

## セットアップ

### 1. 管理画面からデータをアップロード

最初に `admin-dashboard.html` を開いて、以下のデータをアップロードします：

1. **スタイリストCSV** (`stylists.csv`)
   - フォーマット: `姓名,セイメイ,勤務サロン名,メールアドレス,アップロード画像ファイル名,年齢`
   - 例:
     ```csv
     姓名,セイメイ,勤務サロン名,メールアドレス,アップロード画像ファイル名,年齢
     山田太郎,ヤマダタロウ,サロンA,yamada@example.com,hair_design_001.jpg,28
     ```

2. **レビューCSV** (`reviews.csv`)
   - フォーマット: `回答者番号,選択した画像ファイル,コメント,職業,年齢,性別,既婚未婚,子供の有無,女性像,...`
   - 例:
     ```csv
     回答者番号,選択した画像ファイル,コメント,職業,年齢,性別,既婚未婚,子供の有無,女性像
     1,hair_design_001.jpg,とても素敵です,会社員,25,女性,未婚,無,エレガント
     ```

3. **画像ZIPファイル**
   - `hair_design_001.jpg` ~ `hair_design_020.jpg` などを含むZIPファイル
   - ZIPファイルは自動的に解凍され、Base64形式でlocalStorageに保存されます

4. **パスワード設定**
   - オーナーパスワード: オーナーダッシュボードへのログイン用
   - 合言葉: パスワード変更時に必要

### 2. ログイン画面を開く

`index.html` を開いて、以下のいずれかでログインします：

- **スタイリスト**: メールアドレスでログイン → `dashboard.html`
- **オーナー**: "オーナー・教育者の方はこちら" ボタン → パスワード入力 → `owner-dashboard.html`

## 使い方

### スタイリスト向け

1. `index.html` でメールアドレスを入力してログイン
2. ダッシュボードで以下を確認：
   - **評価タブ**: 顧客層分析（年代、職業、女性像など）
   - **レビュータブ**: 自分の作品へのコメント
   - **マイページ**: プロフィール情報

### オーナー向け

1. `index.html` で "オーナー・教育者の方はこちら" をクリック
2. パスワードを入力してログイン
3. ダッシュボードで以下を管理：
   - **概要**: スタッフ統計、トップパフォーマー
   - **スタッフ一覧**: 全スタッフの詳細情報と検索
   - **比較分析**: 最大4人のスタッフを並べて比較
   - **顧客層分析**: 年齢、職業、既婚状況などの統計
   - **ハイライト**: 人気スタッフ、女性像別分析
   - **設定**: パスワード変更

### 管理者向け

1. `admin-dashboard.html` を開く
2. CSV、画像、パスワードをアップロード
3. データ状況を確認
4. 必要に応じてデータをリセット

## セキュリティとプライバシー

- **個人情報の保護**: CSVファイルや画像はコードに含まれません
- **ローカルストレージ**: すべてのデータはブラウザのlocalStorageに保存
- **パスワード管理**: オーナーパスワードと合言葉は管理画面から設定可能
- **GitHub公開可能**: 個人情報を含まないため、安全に公開できます

## 技術スタック

- **フロントエンド**: HTML, CSS, JavaScript
- **チャート**: Chart.js
- **CSV解析**: PapaParse
- **ZIP解凍**: JSZip
- **ストレージ**: Browser localStorage

## ライセンス

このプロジェクトはオープンソースです。自由に使用・改変できます。

## トラブルシューティング

### データが表示されない
- 管理画面（admin-dashboard.html）からCSVと画像をアップロードしてください

### ログインできない
- スタイリスト: メールアドレスがstylists.csvに含まれているか確認
- オーナー: 管理画面からパスワードを設定してください

### 画像が表示されない
- 管理画面から画像ZIPをアップロードしてください
- 画像ファイル名がCSVの「アップロード画像ファイル名」と一致しているか確認

## 開発者向け

### ローカル開発

```bash
# ローカルサーバーを起動（Python 3）
python -m http.server 8000

# ブラウザで開く
open http://localhost:8000
```

### データリセット

管理画面の「データリセット」ボタンをクリックすると、すべてのlocalStorageデータが削除されます。
